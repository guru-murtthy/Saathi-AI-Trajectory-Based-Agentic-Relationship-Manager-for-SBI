# Saathi AI - Relationship Manager Frontend

The React/Vite web application dashboard for relationship managers to inspect customer FFI metrics, look up peer cohort trends, navigate financial projections, and interact with the AI assistant.

## Developer Setup
Run these commands from this directory to start the local development server:
```bash
npm install
npm run dev
```

## Accessibility Intent & Compliance
The Saathi AI dashboard is designed with WCAG AA compliance in mind, maintaining strict color contrast ratios (exceeding 4.5:1) for optimal readability and enabling complete keyboard-tab navigation across all chat functions, input forms, and data panels.
